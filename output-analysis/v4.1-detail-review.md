## Open questions surfaced

- §5.6.3 L:1188's cross-reference exposes a deeper structural gap:
  the spec body has no normative description of the vendor /
  Qualabs URI pattern. `context/06` carries the rule, but
  downstream readers consuming only the spec cannot reconstruct
  it. Worth deciding in the next major build whether the spec
  surfaces this pattern itself or restricts vendor extensions to
  context-defined territory only.
